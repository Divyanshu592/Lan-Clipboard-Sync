# 📡 LAN Clipboard Sync Tool

A simple Python-based tool that allows you to share clipboard content (text) across multiple devices connected to the same WiFi network (LAN), without using the internet.

---

## 🚀 Features

- 📋 Real-time clipboard sync
- 🌐 Works over local network (no internet required)
- 🖥️ Multi-device support
- ⚡ Lightweight and fast
- 🔌 Easy setup

---

## 🧠 How It Works

- One device acts as a **server**
- Other devices act as **clients**
- When clipboard changes on the server, data is sent to all connected clients
- Clients automatically update their clipboard

---

## 🛠️ Tech Stack

- Python
- Socket Programming (TCP)
- Pyperclip (Clipboard access)
